Vitos e-commerce server

Esse é o endereço do servidor do meu projeto pessoal/site: E-commerce.
É através do servidor que faço interações do meu front-end com minha Database, como alimentar meu site com os itens disponíveis na Database, por exemplo.
GET / - Essa página aqui.
GET /itensDaLoja - Itens disponíveis na loja.
POST /efetuarCompra - Efetuar compra através da página de carrinho.
POST /efetuarCompraPeloItem - Efetuar compra através da página dinâmica de algum item.
POST /registerUser - Registro.
POST /logar - Login.
POST /checkJwt - Checagem de JWT token.
POST /editarUser - Editar profile do usuário.
POST /alimentarHistorico - Após logado, utilizado para alimentar a página de compras passadas do usuário.
